## ⚡ Installation

> *"One command. No registration, no SMS, no government approval."*

```bash
curl -fsSL https://raw.githubusercontent.com/NIK-TIGER-BILL/MatryoshkaClaw/main/install.sh | bash
```

That's it. The script checks Node.js, installs deps, builds the project and adds `matryoshka` to PATH.

> ⚠️ **macOS (Homebrew users):**
> ```bash
> SHARP_IGNORE_GLOBAL_LIBVIPS=1 curl -fsSL https://raw.githubusercontent.com/NIK-TIGER-BILL/MatryoshkaClaw/main/install.sh | bash
> ```
> Why? Long story involving libvips, Homebrew and an existential crisis.

### After install

```bash
matryoshka --version      # 🪆 MatryoshkaClaw 2026.x.x
matryoshka setup          # initial configuration
matryoshka onboard        # connect a messenger (try Max, it's patriotic)
matryoshka gateway start  # start the gateway and go make tea
```
