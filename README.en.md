## ⚡ Installation

```bash
curl -fsSL matryoshkaclaw.ru/install | bash
```

One command. No registration, no SMS, no government approval. The script checks Node.js, installs deps, and adds `matryoshka` to PATH.

```bash
matryoshka setup          # initial configuration
matryoshka onboard        # connect a messenger (try Max, it's patriotic)
matryoshka gateway start  # start the gateway and go make tea
```

### Requirements

| What | Version | Note |
|------|---------|------|
| Node.js | ≥ 22 | Yes, exactly 22. 20 is last decade |
| pnpm | ≥ 9 | npm works too, we won't judge |
| git | any | If you don't have git — wrong place |
| Internet | preferred | Agent without internet is just a script |

### Diagnostics

```bash
matryoshka doctor
```

Runs a full environment check. Ends with **Matryoshka Diagnostics** 🩺: a block with Russian-flavored comments for each component.

### Manual install (if curl is scary)

```bash
git clone https://github.com/NIK-TIGER-BILL/MatryoshkaClaw.git
cd MatryoshkaClaw
SHARP_IGNORE_GLOBAL_LIBVIPS=1 npm install -g --force .
matryoshka setup
```

> *If curl is scary — that's normal. The agents are scared too. Just do it.*

</details>
