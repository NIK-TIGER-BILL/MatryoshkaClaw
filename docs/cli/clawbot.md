# `matryoshka clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `matryoshka clawbot qr` (same behavior as [`matryoshka qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `matryoshka clawbot qr` -> `matryoshka qr`
